The micro is now a real-time WebGL device by default: physically-based keycaps over glowing LEDs with true bloom, a knurled reasoning knob, a spring-loaded joystick that quivers on release, studio lighting, and an e-ink readout — rendered with three.js + react-three-fiber on React 19, shipped as a lazy chunk with automatic fallback to the CSS face. Plus bring-your-own-switch acoustics: import a recording of your favorite key switch (press + optional release) as the `custom` sound preset, stored locally.
